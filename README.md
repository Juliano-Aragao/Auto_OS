# 🚗 Sistema de Ordem de Serviço para Oficina Mecânica

## 📖 Sobre o Projeto

O **Sistema de Ordem de Serviço para Oficina Mecânica** foi desenvolvido em **Python** com o objetivo de simular o funcionamento de uma oficina mecânica, permitindo o gerenciamento de clientes, abertura e fechamento de Ordens de Serviço (OS) e controle financeiro básico.

Este projeto foi criado como parte da minha evolução nos estudos de desenvolvimento de software, buscando aplicar conceitos fundamentais de programação, organização de código, tratamento de exceções, manipulação de listas, dicionários e utilização de bibliotecas externas.

Mais do que um exercício, este sistema representa um projeto completo desenvolvido para consolidar conhecimentos em Python antes da migração para banco de dados e desenvolvimento web.

---

# 🚀 Funcionalidades

O sistema possui as seguintes funcionalidades:

* ✅ Cadastro de Clientes
* ✅ Listagem de Clientes
* ✅ Abertura de Ordem de Serviço
* ✅ Consulta de Ordens de Serviço
* ✅ Fechamento de Ordem de Serviço
* ✅ Controle de Caixa
* ✅ Cálculo do faturamento total
* ✅ Cálculo do valor de OS abertas
* ✅ Cálculo do valor de OS fechadas
* ✅ Relatório financeiro por cliente
* ✅ Validação de datas
* ✅ Validação de entradas do usuário
* ✅ Interface amigável no terminal

# 📦 Bibliotecas utilizadas

### Pandas

Utilizado para gerar relatórios financeiros, realizar agrupamentos e cálculos de faturamento.

Instalação:

```bash
pip install pandas
```

---

### Tabulate

Utilizado para apresentar os dados em formato de tabela no terminal.

Instalação:

```bash
pip install tabulate
```

---

### Datetime

Biblioteca nativa do Python utilizada para manipulação e validação de datas.

Não necessita instalação.

---

### OS

Biblioteca nativa utilizada para limpar a tela do terminal de acordo com o sistema operacional.

Não necessita instalação.
